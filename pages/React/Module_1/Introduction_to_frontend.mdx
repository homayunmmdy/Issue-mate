# What is the Libraries , Framwork , Virtual DOM 

## Libraries
- Repository of pre-written code segments
- Provide rusable functions and routines
- Target specific tasks or functionalities
- Offer flexiblitiy for integration
- Examples : jQuery, Lodash , D3.js , React

## Framwork
- A comprehensive software development platform 
- Provide foundation and structure for application building 
- Handle user requests
- Manage data flow
- Impose architecture 
- Examples : Angular.JS , Ember JS , Svelte , Vue.JS

# Virtual DOM
- Generate virtual DOM copies
- Compare to real DOM
- Update essential segments
- Optimize performance
- Minimize DOM modification
